# jMineWASM
C++ Minecraft-like voxel game compiled to WebAssembly using Emscripten.

- World Size is `1024 x 1024` blocks
 

## Play Game
The game is playable via any modern web browser:
[https://jmine.appserver.uk/](https://jmine.appserver.uk/)
